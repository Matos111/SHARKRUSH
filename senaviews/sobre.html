Sobre o Projeto

Inovação em Fitness

O SharkRush é mais do que um simples gerador de treinos - é uma plataforma inteligente que revoluciona a forma como você se exercita. Utilizando algoritmos avançados e uma interface intuitiva, criamos experiências de treino personalizadas que se adaptam ao seu perfil, objetivos e evolução.

Nossa missão é democratizar o acesso a treinos de qualidade profissional, tornando o fitness mais acessível, eficiente e motivador para todos os níveis de condicionamento físico.

Recursos Principais

  • Treinos Personalizados Inteligentes
  • Interface Intuitiva e Moderna
  • Adaptação aos Objetivos do Usuário
  • Monitoramento de Progresso
  • Calculadoras de IMC e Calorias

Linha do Tempo do Projeto

Concepção do Projeto

Identificação da necessidade de uma plataforma inteligente para geração de treinos personalizados, como parte do projeto final do curso de Desenvolvimento de Sistemas do SENAI.

Planejamento e Design

Desenvolvimento da arquitetura do sistema, criação dos wireframes e definição das tecnologias a serem utilizadas para criar uma experiência única ao usuário.

Desenvolvimento

Implementação das funcionalidades principais, incluindo algoritmos de geração de treinos, interface responsiva e sistema de personalização avançado.

Testes e Refinamento

Fase de testes extensivos, otimização de performance e refinamento da experiência do usuário baseado em feedback real.

Lançamento

Entrega do projeto final, apresentação para a banca avaliadora e disponibilização da plataforma para a comunidade fitness.
